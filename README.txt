To run this program do the following:

1) in command prompt cd go into  COMP435 - Assignment Two > comp435 > tma2
2) Here you should see the raw java files.
3) In this directory run the command javac *.java
4) The files are now ready to run, go back two directories so that you're in "COMP435 - Assignment Two"
5) Now input the command java comp435.tma2.COMP435TMA2
6) Once this is done, wait a few seconds and you will be prompted to select a .bmp file
7) Once the file is selected you should be able to see a filled out GUI with options. 
8) When you are prompted to save any files, the files will automatically saved in "COMP435 - Assignment Two"
